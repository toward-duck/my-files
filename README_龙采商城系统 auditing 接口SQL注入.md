# 龙采商城系统 auditing 接口 SQL 注入漏洞检测工具

这是一个基于 Python 编写的自动化多进程安全检测脚本，专门用于检测**龙采商城系统**的 `/coupon/auditing` 接口是否存在基于报错注入（`updatexml`）的 SQL 注入漏洞。

## ✨ 功能特性

* **高精准度检测**：严格比对响应包中的数据库报错特征值（`XPATH syntax error`），有效降低公网扫描中的误报率。
* **原生报文还原**：100% 还原 Burp Suite 验证成功的数据包结构，规避由于 `requests` 自动编码可能导致的 WAF 拦截或解析失效。
* **多进程高速扫描**：内置多进程（`multiprocessing`）并发支持，配备多进程安全锁（`Lock`），在大批量资产检测时保证日志写入不冲突、不丢失。
* **结果自动归档**：检测到的漏洞资产会自动去重并格式化输出到 `vuln.txt`，未受漏洞影响的资产不写入文件，保持结果纯净。
